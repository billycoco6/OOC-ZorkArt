Welcome to Zork and to get you ready for the sake of fun, this file will guide you the compilation and execution.

a. make sure you are at the root of this project ('pwd' should return 'your/path/OOC-ZorkArt')
b. type in the command 'mvn package' to build the project and you will get the folder 'target'
c. 'cd target' to go into the folder
d. 'java -jar Zork-1.0-SNAPSHOT.jar' to compile the project
